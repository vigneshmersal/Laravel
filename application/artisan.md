# PHP artisan
Start development
> php artisan serve

Run php code
> php artisan tinker

Model
> php artisan make:model -mc

Middleware
> php artisan make:middleware CheckAge

Controller
> php artisan make:controller --resource

Database
> php artisan migrate
> php artisan migrate:fresh --seed
> php artisan db:seed

Clear
> php artisan route:clear
> php artisan cache:clear
> php artisan view:clear
