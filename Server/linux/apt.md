# apt

> `man {{apt}}` - help

Instead of *apt-get*, now *apt*,
- `apt install {{package}}`
- `apt remove {{package}}`
- `apt show {{packages}}`
- `apt purge {{packages}}`
- `apt depends {{packages}}`
- `apt policy {{packages}}`
- `apt search {{query}}`
- `apt update`
- `apt upgrade`
- `apt list --installed`
- `apt list --installed`
- `apt edit-sources`
- `apt autoremove`
- `apt autoclean`
